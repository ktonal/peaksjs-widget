
peaksjs_widget
=====================================

Version: |release|

ipywidget to interact with audio waveforms through peaks.js


Quickstart
----------

To get started with peaksjs_widget, install with pip::

    pip install peaksjs_widget

or with conda::

    conda install peaksjs_widget
